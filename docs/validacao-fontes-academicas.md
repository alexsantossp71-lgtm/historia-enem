## Metodologia de Validação

### Critérios Utilizados
1. **Autoridade acadêmica**: Obras de referência reconhecidas
2. **Atualização**: Publicações recentes (últimos 20 anos)
3. **Consensos historiográficos**: Concordância entre fontes
4. **Relevância para ENEM**: Alinhamento com matriz de referência

### Processo de Validação
1. **Pesquisa bibliográfica**: Identificação de fontes primárias
2. **Consulta a especialistas**: Professores universitários
3. **Verificação cruzada**: Comparação entre fontes
4. **Alinhamento curricular**: Compatibilidade com ENEM
